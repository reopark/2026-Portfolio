메인/로그인/다이닝 페이지를 중심으로 정보 구조를 정리하고, 브랜드 톤을 유지한 채 실제 구현 기준으로 UI 시스템과 인터랙션을 정리했습니다

Role UI/UX · Publishing

01

Overview

프리미엄 호텔 브랜드의 ‘여백’과 ‘정돈감’을 유지하면서도, 핵심 진입(예약/탐색)을 더 명확히 만드는 것이 목표였습니다.

Problem

  • 메인사이트에서 제대로 된 정보가 없음
  • 이벤트 , 다이닝 콘텐츠를 확인할려면 여기저기 들어가서 확인해야함
  • 호텔 종류가 많아 하나하나 들어가서 확인해야함

Goal

  • 예약 영역을 메인 중심에 배치해 첫 화면에서 핵심 행동을 유도
  • 타이포·간격·카드 시스템을 재정의해 프리미엄 톤앤매너 강화
  • 탐색 구조를 단순화하여 ‘필터 → 리스트 → 상세’ 흐름으로 개선

메인 이미지와 예약 바(Hotel / Date / Guests)를 중심으로 흐름을 정리했습니다.

02

Process

브랜드의 프리미엄 톤을 유지하면서, 사용자 탐색 흐름을 단순화하는 것을 기준으로 설계했습니다.

Research

프리미엄 호텔 사이트 벤치마킹과 예약 중심 사용자 흐름(CTA) 분석

Structure

메뉴·섹션 정보 구조를 재정리하고 리스트 탐색 흐름을 단순화

UI System

타이포·여백·카드·버튼 규칙을 정리해 일관된 디자인 시스템 구축

Publishing

컴포넌트 단위 퍼블리싱 및 필터·드롭다운 등 인터랙션 구현

04

Result

리뉴얼을 통해 “진입 동선”과 “UI 일관성”을 동시에 개선했습니다.

Key Learnings

  • 예약 CTA를 히어로 중심에 배치했을 때 사용자 행동 유도가 명확해짐
  • 컴포넌트 간 간격·타이포 규칙만 통일해도 브랜드 완성도가 상승함
  • 리스트 UX는 ‘필터 → 결과 → 상세’ 흐름이 탐색 효율에 핵심적임

Next Step

  • 썸네일 인터랙션을 추가해 탐색 경험 개선
  • 반응형 환경에서 예약바를 Step UI 형태로 재구성
  • 다이닝 리스트를 데이터 기반 구조로 확장해 유지보수성 강화
