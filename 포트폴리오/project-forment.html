Forment
Website Renewal

감성적인 브랜드 무드를 유지하면서, 이커머스 탐색·구매 흐름을 더 직관적으로 개선한 리디자인 프로젝트.

Role UI/UX · Publishing
Type Responsive Web
Tools Figma · HTML/CSS · jQuery
01

Overview

이미지 중심 정보 전달과 여백 설계로 브랜드 무드와 탐색 흐름을 동시에 정돈했습니다.

Problem

  • 메인 사이트가 가로 슬라이드 중심이라 핵심 콘텐츠 인지가 느림
  • Best Item 노출 구조가 분산되어 첫 화면에서 제품 매력이 약하게 전달됨
  • 상세 페이지 정보 밀도가 높아 탐색 피로도가 증가

Goal

  • 세로 스크롤 기반 구조로 전환해 콘텐츠 흐름을 자연스럽게 구성
  • 메인 이후 Best Item을 바로 배치해 제품 집중도를 강화
  • 정보를 섹션 단위로 재정렬해 탐색 부담을 줄이고 가독성 개선

메인 > 리스트 > 상세 흐름을 기준으로 전체 경험을 구성했습니다.

Process

리서치부터 UI 시스템 정리까지, 실제 구현을 기준으로 단계별로 진행했습니다.

Research

기존 쇼핑몰 UX 벤치마킹 및 메인 히어로·상품 노출 방식 분석

Structure

메인 → 베스트 → 배너 → 영상 흐름으로 정보 우선순위 재구성

User Flow

세로 스크롤 기반으로 리스트 → 상세 → 구매 흐름 단순화

UI System

타이포·컬러·카드 컴포넌트를 공통화해 화면 일관성 강화

04

Result

브랜드 무드는 유지하면서, 메인페이지와 탐색과 구매 경험을 구조적으로 개선한 것이 핵심 결과입니다.

Key Learnings

  • 정보 구조를 단순화하자 콘텐츠 탐색 흐름이 자연스럽게 이어짐
  • 구매 CTA를 고정 영역에 배치하니 전환 흐름이 명확해짐
  • 여백과 간격을 시스템화하니 전체 완성도가 크게 상승함

Next Step

  • 모바일 상세 페이지에서 구매 버튼 가시성 개선
  • 접근성 기준(포커스·키보드 이동) 반영
  • 컴포넌트 구조를 모듈화해 유지보수 효율 향상
