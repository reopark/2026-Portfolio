01

Overview

브랜드의 스포티한 무드를 유지하면서, Main · MEN 카테고리 · Login 흐름을 재구성해 탐색 구조와 전환 경험을 개선한 리뉴얼 프로젝트입니다.

Problem

  • 메인에서 카테고리와 상품 탐색 흐름이 분리되어 사용자 동선이 끊어짐
  • 상품 리스트가 콘텐츠 사이에 분산되어 쇼핑 집중도가 낮음
  • 브랜드 무드는 강하지만 구매 전환 흐름이 약함

Goal

  • NEW IN · BEST 중심으로 상품 탐색 흐름 재구성
  • 카테고리 → 상품 → 상세로 이어지는 구조 단순화
  • 브랜드 무드는 유지하면서 쇼핑 중심 UI 강화

메인 이미지의 임팩트를 유지하면서, 아래 섹션을 “상품 중심”으로 정리했습니다.

Result

브랜드 무드는 유지하면서, 메인의 탐색 흐름과 UI 규칙을 정돈했습니다.

Key Learnings

  • 쇼핑몰 메인은 브랜드보다 탐색 구조가 먼저 설계되어야 한다
  • 상품 리스트의 순서만 바꿔도 체류 흐름이 달라진다
  • 이미지 중심 UI일수록 여백 규칙이 중요하다
